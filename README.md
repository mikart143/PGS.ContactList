# Friends Contact List
API mainly focused project. Fully validated endpoints. 
Available OData querying: Expand, Select, OrderBy, Count, Filter.
API docs Swagger and ReDoc.
# Used technologies
  - C# 7.0+
  - ASP .Net Core
  - Entity Framework Core 
  - OData
  - NSwag
